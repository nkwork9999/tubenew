初めまして。
このツールについて説明します。
以前のyoutubeではyoutubeのチャンネルごとに過去の動画を一覧ですぐに確認することが出来ていました。
しかし仕様変更によりチャンネル設定によっては過去動画を時系列順に一覧で出すことが出来なくなりました。
またyoutubeのチャンネル内のキーワード検索についてもすぐには見つけられない場所にあり、またその都度入力が必要となります。
そこでこのつべNewを作成しました。
このツールでは指定したチャンネルのyoutube動画を5件取得し、画面にリストとして表示、そのまま再生することが出来ます。
もちろんユーザーごとにリストの作成が可能です。
使用方法について以下に軽く記載します。
⓪まずお手数ですがyoutubeAPIの使用を可能にするために、GoogleCloudへ登録しYoutube Data API v3を有効とします。
そして.envファイル内に　VITE_APP_YOUTUBE_API_KEY = '取得したAPIキーの入力'　の記載を行なってください。
①TOP画面にて登録ボタンを押し、ユーザー登録します。
②ログイン後に青丸の＋ボタンを押すとキーワード等の入力のためのウインドウが開きます。
③そこへタイトル、チャンネルURL(指定したいyoutubeチャンネルのホームのURLを入力)、キーワードを入力してください。※チャンネルURL,キーワードは入力無しでも可。
④youtube内に指定した動画があれば動画が表示されます。
⑤キーワード等を編集したい場合は青ボタン内のリストの内容を編集をクリックし編集してください。
また機能としてリスト同士のドラッグ&ドロップが可能ですが、現状では多用するとyoutube APIの制限がかかりやすくなるため注意してください。
